适用于 Windows 11 / Xbox 的播客应用

Podcasted 是一款沉浸式播客应用，适用于 Windows 通用平台。你可以创建或导入自己的播客库，并在设备之间进行同步。Podcasted 的用户界面现代而极简，有别于传统的 Windows 应用模式。它旨在提供的不仅仅是“工具”，而是“体验”播客的收听方式。

功能亮点：

  • 进度记忆：随时退出，下一次继续播放
  • 订阅与下载：一键订阅并下载播客
  • 变速播放：0.5 倍速至 2 倍速可调
  • 投放播放：支持 Wi-Fi 投屏至其他设备
  • 同步：通过云端同步应用数据
  • 睡眠定时器：让你在设定时间自动停止播放
  • 全平台支持：适用于桌面、平板、手机与 Xbox，覆盖所有 Windows 平台
  • 在线播放：无需下载即可播放节目
  • 音视频支持：支持音频与视频播客
  • 一键刷新：一键检查是否有新节目
  • 筛选功能：按播放状态、下载、时间或字母排序筛选节目
  • 搜索功能：搜索任何你想听的播客，支持节目内搜索
  • 分享：轻松分享播客或具体节目
  • OPML 导入：支持 OPML 导入，迁移更轻松
  • 主题切换：支持深色与浅色模式
  • 离线收听：下载后，无网环境下依然可收听
  • 自动下载：每次启动自动下载最新节目
  • 自动清理：按你设定的数量保留节目，自动管理存储空间
  • 发现频道：发现你可能感兴趣的新播客
  • 播放列表：构建你的专属节目列表
  • 新节目速递：及时获取最新节目
